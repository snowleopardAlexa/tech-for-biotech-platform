Event Space Availability

When "Looking for a job" mode is enabled on your profile, you can see the four leaf clover in the bottom-right corner of your display picture. The clover is visible to all the members of the platform. By default, "Looking for a job" mode is off. Make sure that you turn on the mode only when you seek for a job. In this way, you let the members know that you are ready to get hired.

Please note that if the mode is off, the green clover is not visible on your profile picture. The above dp is just an example how the clover would look like on your profile picture.

On Off

Please note that you should keep the mode on only when look for a job. Otherwise, you might recieve unwanted invitations for job interviews.
